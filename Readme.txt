
To run the code:
	-> Copy the following dataset files into the Code folder.
			Lena.png
			Tag0.mp4
			Tag1.mp4
			Tag2.mp4
			multipleTags.mp4
					
	-> Run Artag.py

	-> Input the video name without file extension which you wish to proccess. 
			Example : For "Tag0.mp4" : Tag0

	-> Select the projection transformation you want to process.
			If you input 1 -> Lena will be projected
			If you input 2 -> Cube will be projected

	-> The code will show frames as it processes and will be generating a video output file as well.
